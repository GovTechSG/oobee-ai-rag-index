# 2.3.3 Animation from Interactions

**Level:** AAA
**Versions:** WCAG 2.1, 2.2
**Principle:** 2 Operable
**Guideline:** 2.3 Seizures and Physical Reactions
**Reference:** [2.3.3 Animation from Interactions](https://www.w3.org/WAI/WCAG22/Understanding/animation-from-interactions.html)

## Description

Motion animation triggered by interaction can be disabled, unless the animation is essential to the functionality or the information being conveyed.

## Sufficient Techniques

- [C39](https://www.w3.org/WAI/WCAG22/Techniques/css/C39): Using the CSS prefers-reduced-motion query to prevent motion
- [SCR40](https://www.w3.org/WAI/WCAG22/Techniques/client-side-script/SCR40): Using the CSS prefers-reduced-motion query in JavaScript to prevent motion
### Gx: Allowing users to set a preference that prevents animation
