# 1.3.5 Identify Input Purpose

**Level:** AA
**Versions:** WCAG 2.1, 2.2
**Principle:** 1 Perceivable
**Guideline:** 1.3 Adaptable
**Reference:** [1.3.5 Identify Input Purpose](https://www.w3.org/WAI/WCAG22/Understanding/identify-input-purpose.html)

## Description

The purpose of each input field collecting information about the user can be programmatically determined when:

## Details

- The input field serves a purpose identified in the Input Purposes for user interface components section; and
- The content is implemented using technologies with support for identifying the expected meaning for form input data.

## Sufficient Techniques

- [H98](https://www.w3.org/WAI/WCAG22/Techniques/html/H98): Using HTML autocomplete attributes

## Failure Techniques

- [F107](https://www.w3.org/WAI/WCAG22/Techniques/failures/F107): Failure of Success Criterion 1.3.5 due to incorrect autocomplete attribute values
