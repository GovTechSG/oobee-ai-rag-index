# 1.2.3 Audio Description or Media Alternative (Prerecorded)

**Level:** A
**Versions:** WCAG 2.0, 2.1, 2.2
**Principle:** 1 Perceivable
**Guideline:** 1.2 Time-based Media
**Reference:** [1.2.3 Audio Description or Media Alternative (Prerecorded)](https://www.w3.org/WAI/WCAG22/Understanding/audio-description-or-media-alternative-prerecorded.html)

## Description

An alternative for time-based media or audio description of the prerecorded video content is provided for synchronized media, except when the media is a media alternative for text and is clearly labeled as such.

## Sufficient Techniques

- [G69](https://www.w3.org/WAI/WCAG22/Techniques/general/G69): Providing an alternative for time based media
  - [G58](https://www.w3.org/WAI/WCAG22/Techniques/general/G58): Placing a link to the alternative for time-based media immediately next to the non-text content
### Linking to the alternative for time-based media


- [G78](https://www.w3.org/WAI/WCAG22/Techniques/general/G78): Providing a second, user-selectable, audio track that includes audio descriptions
- [G173](https://www.w3.org/WAI/WCAG22/Techniques/general/G173): Providing a version of a movie with audio descriptions
  - [SM6](https://www.w3.org/WAI/WCAG22/Techniques/smil/SM6): Providing audio description in SMIL 1.0
  - [SM7](https://www.w3.org/WAI/WCAG22/Techniques/smil/SM7): Providing audio description in SMIL 2.0
  - [G226](https://www.w3.org/WAI/WCAG22/Techniques/general/G226): Providing audio descriptions by incorporating narration in the soundtrack
- [G8](https://www.w3.org/WAI/WCAG22/Techniques/general/G8): Providing a movie with extended audio descriptions
  - [SM1](https://www.w3.org/WAI/WCAG22/Techniques/smil/SM1): Adding extended audio description in SMIL 1.0
  - [SM2](https://www.w3.org/WAI/WCAG22/Techniques/smil/SM2): Adding extended audio description in SMIL 2.0
- [G203](https://www.w3.org/WAI/WCAG22/Techniques/general/G203): Using a static text alternative to describe a talking head video
