# 3.1.6 Pronunciation

**Level:** AAA
**Versions:** WCAG 2.0, 2.1, 2.2
**Principle:** 3 Understandable
**Guideline:** 3.1 Readable
**Reference:** [3.1.6 Pronunciation](https://www.w3.org/WAI/WCAG22/Understanding/pronunciation.html)

## Description

A mechanism is available for identifying specific pronunciation of words where meaning of the words, in context, is ambiguous without knowing the pronunciation.

## Sufficient Techniques

- [G120](https://www.w3.org/WAI/WCAG22/Techniques/general/G120): Providing the pronunciation immediately following the word
- [G121](https://www.w3.org/WAI/WCAG22/Techniques/general/G121): Linking to pronunciations
- [G62](https://www.w3.org/WAI/WCAG22/Techniques/general/G62): Providing a glossary
- [G163](https://www.w3.org/WAI/WCAG22/Techniques/general/G163): Using standard diacritical marks that can be turned off
- [H62](https://www.w3.org/WAI/WCAG22/Techniques/html/H62): Using the ruby element
