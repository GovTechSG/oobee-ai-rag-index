# 1.2.4 Captions (Live)

**Level:** AA
**Versions:** WCAG 2.0, 2.1, 2.2
**Principle:** 1 Perceivable
**Guideline:** 1.2 Time-based Media
**Reference:** [1.2.4 Captions (Live)](https://www.w3.org/WAI/WCAG22/Understanding/captions-live.html)

## Description

Captions are provided for all live audio content in synchronized media.

## Sufficient Techniques

- [G9](https://www.w3.org/WAI/WCAG22/Techniques/general/G9): Creating captions for live synchronized media **AND** - [G93](https://www.w3.org/WAI/WCAG22/Techniques/general/G93): Providing open (always visible) captions
- [G9](https://www.w3.org/WAI/WCAG22/Techniques/general/G9): Creating captions for live synchronized media **AND** - [G87](https://www.w3.org/WAI/WCAG22/Techniques/general/G87): Providing closed captions
