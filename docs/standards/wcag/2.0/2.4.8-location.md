# 2.4.8 Location

**Level:** AAA
**Versions:** WCAG 2.0, 2.1, 2.2
**Principle:** 2 Operable
**Guideline:** 2.4 Navigable
**Reference:** [2.4.8 Location](https://www.w3.org/WAI/WCAG22/Understanding/location.html)

## Description

Information about the user's location within a set of web pages is available.

## Sufficient Techniques

- [G65](https://www.w3.org/WAI/WCAG22/Techniques/general/G65): Providing a breadcrumb trail
- [G63](https://www.w3.org/WAI/WCAG22/Techniques/general/G63): Providing a site map
- [G128](https://www.w3.org/WAI/WCAG22/Techniques/general/G128): Indicating current location within navigation bars
- [G127](https://www.w3.org/WAI/WCAG22/Techniques/general/G127): Identifying a web page's relationship to a larger collection of web pages

## Advisory Techniques

- [PDF14](https://www.w3.org/WAI/WCAG22/Techniques/pdf/PDF14): Providing running headers and footers in PDF documents
- [PDF17](https://www.w3.org/WAI/WCAG22/Techniques/pdf/PDF17): Specifying consistent page numbering for PDF documents
