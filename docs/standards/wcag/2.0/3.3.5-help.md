# 3.3.5 Help

**Level:** AAA
**Versions:** WCAG 2.0, 2.1, 2.2
**Principle:** 3 Understandable
**Guideline:** 3.3 Input Assistance
**Reference:** [3.3.5 Help](https://www.w3.org/WAI/WCAG22/Understanding/help.html)

## Description

Context-sensitive help is available.

## Sufficient Techniques

### Situation A: If a form requires text input:

- [G71](https://www.w3.org/WAI/WCAG22/Techniques/general/G71): Providing a help link on every web page
- [G193](https://www.w3.org/WAI/WCAG22/Techniques/general/G193): Providing help by an assistant in the web page
- [G194](https://www.w3.org/WAI/WCAG22/Techniques/general/G194): Providing spell checking and suggestions for text input
- [G184](https://www.w3.org/WAI/WCAG22/Techniques/general/G184): Providing text instructions at the beginning of a form or set of fields that describes the necessary input

### Situation B: If a form requires text input in an expected data format:

- [G89](https://www.w3.org/WAI/WCAG22/Techniques/general/G89): Providing expected data format and example
- [G184](https://www.w3.org/WAI/WCAG22/Techniques/general/G184): Providing text instructions at the beginning of a form or set of fields that describes the necessary input


## Advisory Techniques

- [H89](https://www.w3.org/WAI/WCAG22/Techniques/html/H89): Using the title attribute to provide context-sensitive help
