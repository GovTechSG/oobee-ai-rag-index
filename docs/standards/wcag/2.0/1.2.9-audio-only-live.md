# 1.2.9 Audio-only (Live)

**Level:** AAA
**Versions:** WCAG 2.0, 2.1, 2.2
**Principle:** 1 Perceivable
**Guideline:** 1.2 Time-based Media
**Reference:** [1.2.9 Audio-only (Live)](https://www.w3.org/WAI/WCAG22/Understanding/audio-only-live.html)

## Description

An alternative for time-based media that presents equivalent information for live audio-only content is provided.

## Sufficient Techniques

- [G151](https://www.w3.org/WAI/WCAG22/Techniques/general/G151): Providing a link to a text transcript of a prepared statement or script if the script is followed
- [G150](https://www.w3.org/WAI/WCAG22/Techniques/general/G150): Providing text based alternatives for live audio-only content
- [G157](https://www.w3.org/WAI/WCAG22/Techniques/general/G157): Incorporating a live audio captioning service into a web page
