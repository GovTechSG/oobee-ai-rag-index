# 2.4.2 Page Titled

**Level:** A
**Versions:** WCAG 2.0, 2.1, 2.2
**Principle:** 2 Operable
**Guideline:** 2.4 Navigable
**Reference:** [2.4.2 Page Titled](https://www.w3.org/WAI/WCAG22/Understanding/page-titled.html)

## Description

Web pages have titles that describe topic or purpose.

## Sufficient Techniques

- [G88](https://www.w3.org/WAI/WCAG22/Techniques/general/G88): Providing descriptive titles for web pages **AND** - associating a title with a web page

## Advisory Techniques

- [G127](https://www.w3.org/WAI/WCAG22/Techniques/general/G127): Identifying a web page's relationship to a larger collection of web pages

## Failure Techniques

- [F25](https://www.w3.org/WAI/WCAG22/Techniques/failures/F25): Failure of Success Criterion 2.4.2 due to the title of a web page not identifying the contents
