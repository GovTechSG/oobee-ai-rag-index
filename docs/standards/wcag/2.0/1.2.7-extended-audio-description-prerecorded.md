# 1.2.7 Extended Audio Description (Prerecorded)

**Level:** AAA
**Versions:** WCAG 2.0, 2.1, 2.2
**Principle:** 1 Perceivable
**Guideline:** 1.2 Time-based Media
**Reference:** [1.2.7 Extended Audio Description (Prerecorded)](https://www.w3.org/WAI/WCAG22/Understanding/extended-audio-description-prerecorded.html)

## Description

Where pauses in foreground audio are insufficient to allow audio descriptions to convey the sense of the video, extended audio description is provided for all prerecorded video content in synchronized media.

## Sufficient Techniques

- [G8](https://www.w3.org/WAI/WCAG22/Techniques/general/G8): Providing a movie with extended audio descriptions
  - [SM1](https://www.w3.org/WAI/WCAG22/Techniques/smil/SM1): Adding extended audio description in SMIL 1.0
  - [SM2](https://www.w3.org/WAI/WCAG22/Techniques/smil/SM2): Adding extended audio description in SMIL 2.0

## Advisory Techniques

- [H96](https://www.w3.org/WAI/WCAG22/Techniques/html/H96): Using the track element to provide audio descriptions
