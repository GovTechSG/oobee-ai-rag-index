# 3.1.3 Unusual Words

**Level:** AAA
**Versions:** WCAG 2.0, 2.1, 2.2
**Principle:** 3 Understandable
**Guideline:** 3.1 Readable
**Reference:** [3.1.3 Unusual Words](https://www.w3.org/WAI/WCAG22/Understanding/unusual-words.html)

## Description

A mechanism is available for identifying specific definitions of words or phrases used in an unusual or restricted way, including idioms and jargon.

## Sufficient Techniques

### Situation A: If the word or phrase has a unique meaning within the web page:

- [G101](https://www.w3.org/WAI/WCAG22/Techniques/general/G101): Providing the definition of a word or phrase used in an unusual or restricted way
  - [G55](https://www.w3.org/WAI/WCAG22/Techniques/general/G55): Linking to definitions
  - [G112](https://www.w3.org/WAI/WCAG22/Techniques/general/G112): Using inline definitions
- [G101](https://www.w3.org/WAI/WCAG22/Techniques/general/G101): Providing the definition of a word or phrase used in an unusual or restricted way
  - [G55](https://www.w3.org/WAI/WCAG22/Techniques/general/G55): Linking to definitions
  - [G62](https://www.w3.org/WAI/WCAG22/Techniques/general/G62): Providing a glossary
  - [G70](https://www.w3.org/WAI/WCAG22/Techniques/general/G70): Providing a function to search an online dictionary

### Situation B: If the word or phrase means different things within the same web page:

- [G101](https://www.w3.org/WAI/WCAG22/Techniques/general/G101): Providing the definition of a word or phrase used in an unusual or restricted way
  - [G55](https://www.w3.org/WAI/WCAG22/Techniques/general/G55): Linking to definitions
  - [G112](https://www.w3.org/WAI/WCAG22/Techniques/general/G112): Using inline definitions
