# 1.2.8 Media Alternative (Prerecorded)

**Level:** AAA
**Versions:** WCAG 2.0, 2.1, 2.2
**Principle:** 1 Perceivable
**Guideline:** 1.2 Time-based Media
**Reference:** [1.2.8 Media Alternative (Prerecorded)](https://www.w3.org/WAI/WCAG22/Understanding/media-alternative-prerecorded.html)

## Description

An alternative for time-based media is provided for all prerecorded synchronized media and for all prerecorded video-only media.

## Sufficient Techniques

### Situation A: If the content is prerecorded synchronized media:

- [G69](https://www.w3.org/WAI/WCAG22/Techniques/general/G69): Providing an alternative for time based media
  - [G58](https://www.w3.org/WAI/WCAG22/Techniques/general/G58): Placing a link to the alternative for time-based media immediately next to the non-text content
- Linking to the alternative for time-based media
  - [H53](https://www.w3.org/WAI/WCAG22/Techniques/html/H53): Using the body of the object element

### Situation B: If the content is prerecorded video-only:

- [G159](https://www.w3.org/WAI/WCAG22/Techniques/general/G159): Providing an alternative for time-based media for video-only content


## Failure Techniques

- [F74](https://www.w3.org/WAI/WCAG22/Techniques/failures/F74): Failure of Success Criterion 1.2.2 and 1.2.8 due to not labeling a synchronized media alternative to text as an alternative
