# 3.1.2 Language of Parts

**Level:** AA
**Versions:** WCAG 2.0, 2.1, 2.2
**Principle:** 3 Understandable
**Guideline:** 3.1 Readable
**Reference:** [3.1.2 Language of Parts](https://www.w3.org/WAI/WCAG22/Understanding/language-of-parts.html)

## Description

The human language of each passage or phrase in the content can be programmatically determined except for proper names, technical terms, words of indeterminate language, and words or phrases that have become part of the vernacular of the immediately surrounding text.

## Sufficient Techniques

- [H58](https://www.w3.org/WAI/WCAG22/Techniques/html/H58): Using language attributes to identify changes in the human language
- [PDF19](https://www.w3.org/WAI/WCAG22/Techniques/pdf/PDF19): Specifying the language for a passage or phrase with the Lang entry in PDF documents
