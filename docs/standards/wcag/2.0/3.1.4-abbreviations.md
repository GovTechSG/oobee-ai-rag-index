# 3.1.4 Abbreviations

**Level:** AAA
**Versions:** WCAG 2.0, 2.1, 2.2
**Principle:** 3 Understandable
**Guideline:** 3.1 Readable
**Reference:** [3.1.4 Abbreviations](https://www.w3.org/WAI/WCAG22/Understanding/abbreviations.html)

## Description

A mechanism for identifying the expanded form or meaning of abbreviations is available.

## Sufficient Techniques

### Situation A: If the abbreviation has only one meaning within the web page:

- [G102](https://www.w3.org/WAI/WCAG22/Techniques/general/G102): Providing the expansion or explanation of an abbreviation
  - [G97](https://www.w3.org/WAI/WCAG22/Techniques/general/G97): Providing the first use of an abbreviation immediately before or after the expanded form
  - [G55](https://www.w3.org/WAI/WCAG22/Techniques/general/G55): Linking to definitions
  - [PDF8](https://www.w3.org/WAI/WCAG22/Techniques/pdf/PDF8): Providing definitions for abbreviations via an E entry for a structure element
- [G102](https://www.w3.org/WAI/WCAG22/Techniques/general/G102): Providing the expansion or explanation of an abbreviation
  - [G55](https://www.w3.org/WAI/WCAG22/Techniques/general/G55): Linking to definitions
  - [G62](https://www.w3.org/WAI/WCAG22/Techniques/general/G62): Providing a glossary
  - [G70](https://www.w3.org/WAI/WCAG22/Techniques/general/G70): Providing a function to search an online dictionary
  - [PDF8](https://www.w3.org/WAI/WCAG22/Techniques/pdf/PDF8): Providing definitions for abbreviations via an E entry for a structure element

### Situation B: If the abbreviation means different things within the same web page:

- [G102](https://www.w3.org/WAI/WCAG22/Techniques/general/G102): Providing the expansion or explanation of an abbreviation
  - [G55](https://www.w3.org/WAI/WCAG22/Techniques/general/G55): Linking to definitions
  - [PDF8](https://www.w3.org/WAI/WCAG22/Techniques/pdf/PDF8): Providing definitions for abbreviations via an E entry for a structure element
