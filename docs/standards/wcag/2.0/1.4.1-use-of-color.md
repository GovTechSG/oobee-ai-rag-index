# 1.4.1 Use of Color

**Level:** A
**Versions:** WCAG 2.0, 2.1, 2.2
**Principle:** 1 Perceivable
**Guideline:** 1.4 Distinguishable
**Reference:** [1.4.1 Use of Color](https://www.w3.org/WAI/WCAG22/Understanding/use-of-color.html)

## Description

Color is not used as the only visual means of conveying information, indicating an action, prompting a response, or distinguishing a visual element.

## Details

> **Note:** This success criterion addresses color perception specifically. Other forms of perception are covered in Guideline 1.3 including programmatic access to color and other visual presentation coding.

## Sufficient Techniques

### Situation A: If the color of particular words, backgrounds, or other content is used to indicate information:

- [G14](https://www.w3.org/WAI/WCAG22/Techniques/general/G14): Ensuring that information conveyed by color differences is also available in text
- [G205](https://www.w3.org/WAI/WCAG22/Techniques/general/G205): Including a text cue for colored form control labels
- [G182](https://www.w3.org/WAI/WCAG22/Techniques/general/G182): Ensuring that additional visual cues are available when text color differences are used to convey information
- [G183](https://www.w3.org/WAI/WCAG22/Techniques/general/G183): Using a contrast ratio of 3:1 with surrounding text and providing additional visual cues on hover for links or controls where color alone is used to identify them

### Situation B: If color is used within an image to convey information:

- [G111](https://www.w3.org/WAI/WCAG22/Techniques/general/G111): Using color and pattern
- [G14](https://www.w3.org/WAI/WCAG22/Techniques/general/G14): Ensuring that information conveyed by color differences is also available in text


## Advisory Techniques

- [C15](https://www.w3.org/WAI/WCAG22/Techniques/css/C15): Using CSS to change the presentation of a user interface component when it receives focus

## Failure Techniques

- [F13](https://www.w3.org/WAI/WCAG22/Techniques/failures/F13): Failure of Success Criterion 1.1.1 and 1.4.1 due to having a text alternative that does not include information that is conveyed by color differences in the image
- [F73](https://www.w3.org/WAI/WCAG22/Techniques/failures/F73): Failure of Success Criterion 1.4.1 due to creating links that are not visually evident without color vision
- [F81](https://www.w3.org/WAI/WCAG22/Techniques/failures/F81): Failure of Success Criterion 1.4.1 due to identifying required or error fields using color differences only
